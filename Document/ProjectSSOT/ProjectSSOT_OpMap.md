# ProjectSSOT 문서 색인

## 목적
이 문서는 `Document/ProjectSSOT/`의 **빠른 탐색용 색인**이다.

- 운영 기준의 원본은 `README.md`를 우선한다.
- 이 문서는 폴더 안에서 어떤 문서를 어디에 써야 하는지 빠르게 찾을 때만 사용한다.

---

## 핵심 문서
- `README.md`
  - 운영 기준 / 문서 역할 / Archive 정책
- `00_Handover.md`
  - 현재 상태 / 확정 기준 / 리스크
- `01_Roadmap.md`
  - 현재 할 일 / 우선순위 / 완료 조건
- `02_Conventions.md`
  - 프로젝트 특화 규칙
- `08_P0_Verification.md`
  - 현재 기준 검증 판정

## 사용 규칙
- 현재 기준을 알고 싶으면 `README.md`와 `00_Handover.md`를 먼저 본다.
- 우선순위를 알고 싶으면 `01_Roadmap.md`를 본다.
- PASS/FAIL 판정은 `08_P0_Verification.md`를 본다.
- 완료된 작업 문서는 루트에 오래 두지 않고 `Archive/`로 이동한다.

# ProjectSSOT Archive 운영 가이드

## 목적
`Document/ProjectSSOT/Archive/`는 CarFight 프로젝트의 **완료된 작업 문서 / 과도기 기록 / 체크포인트 로그 / 문서 운영 기록**을 보관한다.

현재 기준 문서는 루트의 활성 문서를 우선하고, Archive 문서는 역사 기록 / 추적 / 회고 목적으로만 사용한다.

---

## 하위 분류
### `Archive/LegacyBP/`
- 구형 `BP_ModularVehicle` 기준 절차
- 구형 BP 기반 핫픽스 / 조사 기록

### `Archive/CPP/`
- 완료된 C++ 전환 단계 문서
- helper/debug pipe 비교 기록
- 단일 축 전환 테스트 기록

### `Archive/Checkpoints/`
- 일자별 SSOT 체크포인트 로그

### `Archive/Ops/`
- 문서 정리 / 이동 실행 기록

---

## 사용 원칙
- Archive 문서는 현재 기준 문서가 아니다.
- 현재 상태 / 우선순위 / 검증 판정은 루트 활성 문서를 먼저 본다.
- Archive 문서 상단에는 가능하면 현재 기준 문서 경로를 함께 남긴다.

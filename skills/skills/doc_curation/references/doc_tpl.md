# 문서 정제 출력 템플릿

이 템플릿은 문서 본문 자체를 처음부터 작성하기 위한 템플릿이 아니라,
기존 문서를 최소 수정 원칙으로 정제한 결과를 정리해서 보여줄 때 사용하는 출력 템플릿이다.

목표는 문서를 무조건 짧게 만드는 것이 아니라,
핵심 요구사항, 제약사항, 결정 이유, 예외 조건, 검증 기준, 작업 맥락을 유지한 채
중복을 줄이고 구조와 가독성을 개선하는 것이다.

문서 성격에 따라 불필요한 섹션은 생략할 수 있다.

## 정제된 문서 또는 수정 구간

여기에 수정된 문서 전체 또는 수정된 구간만 넣는다.

- 전체 문서를 다시 보여줄 필요가 없다면 변경된 구간만 제시한다.
- 가능한 경우 수정 범위가 드러나도록 섹션 단위로 제시한다.
- 구조 이동이 있었다면 이동된 위치가 드러나게 적는다.

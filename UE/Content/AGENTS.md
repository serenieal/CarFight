이 경로 이하에서는 블루프린트와 에셋 작업 규칙을 따른다.

실제로 존재하는 블루프린트 노드 이름만 사용한다.
개념적인 표현을 실제 노드처럼 설명하지 않는다.
노드나 기능의 존재가 확실하지 않으면 추측하지 않고 불확실하다고 명시한다.

블루프린트 변수와 함수에는 이해하기 쉬운 툴팁 문구를 함께 제안한다.
변수명과 함수명은 직관적으로 짓는다.
무분별한 에셋 리네이밍과 경로 변경은 금지한다.

블루프린트 설명 시:
- 어느 블루프린트에서 작업하는지 먼저 적는다.
- 변수 추가/수정 여부를 분리해서 적는다.
- 함수 추가/수정 여부를 분리해서 적는다.
- 검증 방법과 예상 결과를 함께 적는다.

디자이너가 읽는 UI, 툴팁, 노출 문자열은 한글로 제안한다.

## 이성수준 기준
- 낮음: 블루프린트/에셋 위치 찾기, 변수/함수/툴팁 정리, 짧은 설명
- 중간: 일반적인 블루프린트 로직 수정, UI 수정, 입력/데이터 연결 수정
- 높음: 재현이 어려운 블루프린트 버그, 레퍼런스 영향이 큰 변경, 다수 에셋에 걸친 영향 분석
- 에셋 경로 변경이나 레퍼런스 검토가 크면 보수적으로 높음으로 올린다.
